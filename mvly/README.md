# Apps

Apps is a container main application that showcases the sub project apps.
Each app has its own sandbox kind of environment on top of the domain home
page.

## Products
- [App Engine][1]

## Language
- [Python][2]

## APIs
- [NDB Datastore API][3]

## Dependencies
- [webapp2][5]
- [jinja2][6]
- [Twitter Bootstrap][7]
